## Description

CurseForge’s "Last Updated" sorting is global — it reflects the most recent update across *any* version of a project. This makes it difficult to find mods or resource packs that are actually up to date for a specific Minecraft version or mod loader.

This program solves that problem.

## Features

* Search mods and resource packs using CurseForge data
* Filter by:
  * Minecraft version
  * Mod loader (Forge, Fabric, etc.)
* Sort results by the last update **for the selected version and loader only**

## Requirements

* CurseForge API key
* Get one by signing up as a developer: [https://console.curseforge.com](https://console.curseforge.com)

## Installation

1. Go to the Releases page
2. Download the appropriate version for your system:
   * `.exe` for Windows
   * Linux binary for Linux
3. Run the downloaded file
